##  QR Code Implementation

A QR code was generated to provide users with fast and convenient access to the deployed BC Wild Watch web application. The QR code removes the need for manual URL entry by allowing users to scan and instantly navigate to the system using a mobile device.

The deployment process involved hosting the application on **Netlify**, which generated a publicly accessible URL. This URL was then converted into a QR code using **QR.io**. During generation, the URL was encoded into a two-dimensional matrix consisting of black and white modules that can be interpreted by QR-enabled devices.

### QR Code Workflow

| Step | Description                           |
| ---- | ------------------------------------- |
| 1    | Deploy application to Netlify         |
| 2    | Obtain generated deployment URL       |
| 3    | Generate QR code using QR.io          |
| 4    | Apply error-correction encoding       |
| 5    | Export QR code as an image            |
| 6    | Share URL and QR code with users      |
| 7    | Users scan and access the application |
